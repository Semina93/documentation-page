Introduction

Flexbox is a one-dimensional layout method for arranging items in rows or columns. For a long time, the only tools available for creating CSS layouts were features like floats and positioning. These work, but in some ways they're also limiting and complex.

The Flexbox Layout Module makes it easier to design responsive layout without using float or positioning. In this documentation we are going to deal with:

  • Flexbox Properties
  • Flex Direction
  • Flex Wrap
  • Justify Content
display: flex;
Flexbox Properties

To start using the Flexbox model, you need to first define a flex container.

The flex container contains the flex items.

Flex Direction

The flex-direction property defines in which direction the container has to stack the flex items.

The values that this property can take are the following:

  • row (deafault)
  • row-reverse
  • column
  • column-reverse
flex-direction: row;
Flex Wrap

The flex-wrap property specifies whether the flex items should wrap or not.

This propetry takes three values:

  • The nowrap value which that the flex items will not wrap (default).
  • The wrap value specifies that the flex items will wrap if necessary.
  • The wrap-reverse value specifies that the flex items will wrap if necessary but from bottom to top.
flex-wrap: wrap;
Justify Content

This defines the horizontal alignment (along the main axis).

This property takes the following values:

  • The center value aligns the flex items at the center of the container.
  • The flex-start value aligns the flex items at the beginning of the container (this is default).
  • The flex-end value aligns the flex items at the end of the container.
  • The space-around value displays the flex items with space before, between, and after the lines.
  • The space-between value displays the flex items with space between the lines.
justify-content: center;
